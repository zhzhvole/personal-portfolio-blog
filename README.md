# personal-portfolio-blog
A personal portfolio &amp; technical blog built with React + Node.js + MongoDB, showcasing my projects, skills and technical articles.
# 个人技术博客与作品集

一个展示我的项目、技能和技术文章的平台，用主流技术开发，适配手机和电脑。

## 核心功能
- 展示我的项目：每个项目都有介绍、用到的技术和在线演示链接
- 技术博客：写技术文章，支持代码高亮、搜索和分类
- 响应式设计：手机、平板、电脑上都能正常显示
- 后台管理：可以自己添加/修改项目和文章

## 用到的技术
- 前端：React（页面开发）、Tailwind CSS（样式设计）、TypeScript（代码规范）
- 后端：Node.js（服务器）、MongoDB（数据库）、JWT（身份验证）
- 工具：Git/GitHub（版本管理）、Vercel（在线部署）、Docker（容器化）

## 如何运行
1. 复制仓库：`git clone https://github.com/你的GitHub用户名/personal-portfolio-blog.git`
2. 安装依赖：`npm install`
3. 配置环境变量（连接数据库）
4. 运行：`npm run dev`，打开浏览器访问 `http://localhost:5173`
